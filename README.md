# Echo Protocol

Real-time encrypted communication system simulation. End-to-end encryption visualization with audio wave visualization of messages.

## Tech Stack

- Next.js
- TypeScript
- Framer Motion
- Tailwind CSS
- WebGL (Audio Viz)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the demo.

## Features

- **Packet Visualizer**: Real-time visualization of encrypted data packets
- **Message Terminal**: Secure messaging interface with encryption toggle
- **P2P Mesh Network**: Simulated peer-to-peer encrypted communication
- **Protocol Specs**: AES-256-GCM encryption, X25519 key exchange, Ed25519 signatures
